## Шаг 2 — Установка на VPS сервере (Ubuntu 22)

Подключись к серверу по SSH и выполни по очереди:

```bash
# Обновить систему
sudo apt update && sudo apt upgrade -y

# Установить Python
sudo apt install python3 python3-pip -y

# Создать папку для бота
mkdir ~/nasiya_bot && cd ~/nasiya_bot

# Загрузить файлы (скопируй все файлы из архива в эту папку)
# Можно через FileZilla или WinSCP

# Установить зависимости
pip3 install -r requirements.txt
```

---

## Шаг 3 — Запустить бота 24/7 через systemd

```bash
# Создать сервис
sudo nano /etc/systemd/system/nasiya_bot.service
```

Вставь этот текст (замени YOUR_USERNAME на имя пользователя сервера):
```ini
[Unit]
Description=Nasiya Baza Bot
After=network.target

[Service]
WorkingDirectory=/home/YOUR_USERNAME/nasiya_bot
ExecStart=/usr/bin/python3 bot.py
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
```

```bash
# Включить и запустить
sudo systemctl daemon-reload
sudo systemctl enable nasiya_bot
sudo systemctl start nasiya_bot

# Проверить статус
sudo systemctl status nasiya_bot
```
